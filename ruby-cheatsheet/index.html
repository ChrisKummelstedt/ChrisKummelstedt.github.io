Array

index = n-1, as such: 

my_array = [A, B, C]  
my_array[0] => A  
my_array[-1] => C  
my_array[0..1] => [1, B

it also works the other way around: 

my_array[-3] => A  
my_array[-4] => nil

Iterating over arrays

.each: do something to each element

my_array = [0, 1, 2]  
my_array.each { |variable| method }

.map: do something to each element and map a new array in relation to the old array

my_array = [0, 1, 2]  
my_new_array = my_array.map { |variable| method }

.each_slice(n): do something to groups of n variables (but don't map it to a new array)

result = []  
my_array = [0, 1, 2, 3]  
my_array.each_slice { |a, b| result << [a, b]}  
result => [[0, 1],[2, 3]]

.each_cons(n): do something to each variables and groups of n consecutive variables (but don't map it to a new array)

result = []  
my_array = [0, 1, 2, 3]  
my_array.each_cons { |a, b| result << [a, b]}  
result => [[0, 1],[1, 2]...]

.select: create a new array for any element that passes a test and returns true 

my_array = [0, 1, 2, 3]  
my_new_array = my_array.select { |number| number.even?}  
my_new_array => [0, 2]

.reject: the opposite of select (if returns true does not include)

my_array = [0, 1, 2, 3]  
my_new_array = my_array.reject { |number| number.even?}  
my_new_array => [1, 3]

If you want to modify the old array, use bang (!) i.e. 

my_array = [0, 1, 2, 3]  
my_array.reject! { |number| number.even?}  
my_array => [1, 3]

Instance variables

Basics: 

true.class => TrueClass  
false.class => FalseClass

Cool way to use this for storing data in a more pleasing way:

.even? *Notice the name of the methods here (ruby knows that method=(blue) is the same as method = blue).

Now, you can do this with a class method called attr_accessor:

class Horse  
 attr_accessor 'color'
end

fluffy = Horse.new  
fluffy.color = "Brown"  
fluffy.color => Brown

You can also set default values to instance variables

class Horse  
 def initalize(color, age)
     @color = nil 
     @age = 0  
 end

 attr_accessor 'color'
end

fluffy = Horse.new 'Brown, 3'

fluffy.color => Brown  
fluffy.age => 3
